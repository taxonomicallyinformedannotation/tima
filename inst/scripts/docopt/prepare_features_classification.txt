You can use this script with the following example:
  Rscript inst/scripts/prepare_features_classification.R --fil-ann-fil data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_filled.tsv.gz --fil-ann-tre data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_treated.tsv.gz --fil-lib-sop-mer data/interim/libraries/library.tsv.gz --fast true

Usage: 
  prepare_features_classification.R [--fil-ann-fil=<fil-ann-fil>] [--fil-ann-tre=<fil-ann-tre>] [--fil-lib-sop-mer=<fil-lib-sop-mer>] [--fast=<fast>]

Arguments:
  fil-ann-fil                Filled annotations file. STRING
  fil-ann-tre                Treated annotations file. STRING
  fil-lib-sop-mer            Merged structure organism pairs library. STRING

  fast                       Skip time-consuming steps. BOOLEAN

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
