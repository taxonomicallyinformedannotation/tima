You can use this script with the following example:
  Rscript inst/scripts/prepare_gnps.R --fil-ann-pre data/interim/96fa7c88200e4a03bee4644e581e3fb0_gnps_pretreated.tsv.gz --gnps-id 96fa7c88200e4a03bee4644e581e3fb0 --gnps-workflow fbmn

Usage: 
  prepare_gnps.R [--=fil-ann-pre<fil-ann-pre>] [--gnps-id=<gnps-id>] [--gnps-workflow=<gnps-workflow>] [--gnps-nap=<gnps-nap>]

Arguments:
  fil-ann-pre                List of pretreated annotations. STRING
  
  gnps-id                    GNPS job ID. STRING
  gnps-workflow              Workflow used within GNPS. Currently "fbmn" or "classical" supported. STRING
  gnps-nap                   OPTIONAL. If exists. NAP job ID. STRING

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
