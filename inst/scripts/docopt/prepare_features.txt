You can use this script with the following example:
  Rscript inst/scripts/prepare_features.R --fil-fea-raw data/source/example_features.csv --fil-fea-pre data/interim/features/example_features.tsv.gz --names-features 'row ID' --names-precursor 'row m/z' --names-rt 'row retention time'

Usage: 
  prepare_features.R [--fil-fea-raw=<fil-fea-raw>] [--fil-fea-pre=<fil-fea-pre>] [--names-features=<names-features>] [--names-precursor=<names-precursor>] [--names-rt=<names-rt>]

Arguments:
  fil-fea-raw                Raw features file. STRING
  fil-fea-pre                Prepared features file. STRING

  names-features             Name of "feature IDs" variable in the input. STRING
  names-precursor            Name of "precursor m/z" variable in the input. STRING
  names-rt                   Name of "retention time" variable in the input. STRING

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
