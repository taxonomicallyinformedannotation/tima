You can use this script with the following example:
  Rscript inst/scripts/annotate_masses.R

Usage:
  annotate_masses.R [--fil-ann-pre-str=<fil-ann-pre-str>] [--fil-lib-sop-mer=<fil-lib-sop-mer>] [--fil-lib-add-neg=<fil-lib-add-neg>] [--fil-lib-add-pos=<fil-lib-add-pos>] [--fil-lib-add-pro=<fil-lib-add-pro>] [--ms-add-neg=<ms-add-neg>] [--ms-add-pos=<ms-add-pos>]  [--ms-add-con-type=<ms-add-con-type>] [--ms-add-con-min-degree=<ms-add-con-min-degree>] [--ms-add-con-min-support=<ms-add-con-min-support>] [--ms-clu=<ms-clu>] [--ms-sol=<ms-sol>] [--ms-neu=<ms-neu>] [--ms-pol=<ms-pol>] [--ms-tol-mas-ppm-ms1=<ms-tol-mas-ppm-ms1>] [--ms-tol-mas-dal-ms1=<ms-tol-mas-dal-ms1>] [--ms-tol-rt-add=<ms-tol-rt-add>] [--ms-tol-int-add=<ms-tol-int-add>] [--force=<force>]

Arguments:
  fil-ann-pre-str            List of prepared structural annotations. STRING
  fil-fea-pre                Prepared features file. STRING
  fil-lib-add-neg            Negative adducts. STRING
  fil-lib-add-pos            Positive adducts. STRING
  fil-lib-add-pro            Prepared adducts library. STRING
  fil-lib-sop-mer            Merged structure organism pairs library. STRING

  ms-add-neg                 List of adducts to be used in negative. STRING
  ms-add-pos                 List of adducts to be used in positive. STRING
  ms-add-con-type            Adduct consistency strategy. Must be one of: off, conditional, strict. STRING
  ms-add-con-min-degree      Minimum feature degree threshold to trigger consistency checks in conditional mode. INTEGER
  ms-add-con-min-support     Minimum number of independent neighbor supports for an adduct assignment. INTEGER
  ms-clu                     List of clusters to be used. Comma-separated STRING
  ms-sol                     List of solvents to be used. Comma-separated STRING
  ms-neu                     List of neutral losses to be used. STRING
  ms-pol                     Polarity used. Must be "pos" or "neg". STRING
  ms-tol-mas-ppm-ms1         Relative mass tolerance for MS1 in ppm. FLOAT
  ms-tol-mas-dal-ms1         Absolute mass tolerance for MS1 in Dalton. FLOAT
  ms-tol-rt-add              Retention time tolerance for adducts in minutes. FLOAT
  ms-tol-int-add             Minimum intensity co-variance correlation for adducts. FLOAT

  force                      Do not use it. BOOLEAN

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
