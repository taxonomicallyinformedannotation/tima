You can use this script with the following example:
  Rscript inst/scripts/prepare_libraries_adducts.R --fil-lib-sop-mer data/interim/libraries/library.tsv.gz --fil-lib-add-pro library

Usage: 
  prepare_libraries_adducts.R [--fil-lib-sop-mer=<fil-lib-sop-mer>] [--fil-lib-add-pro=<fil-lib-add-pro>]

Arguments:
  fil-lib-add-pro            Prepared adducts library. STRING
  fil-lib-sop-mer            Merged structure organism pairs library. STRING

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
