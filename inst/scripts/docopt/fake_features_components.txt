  Rscript inst/scripts/fake_features_components.R --fil-ann-pre data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_gnps_pretreated.tsv.gz --fil-ann-pre data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_isdb_pretreated.tsv.gz --fil-ann-pre data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_sirius_pretreated.tsv.gz --fil-fea-raw data/interim/annotations/example_feature_table.tsv.gz --fil-ann-fil data/interim/annotations/96fa7c88200e4a03bee4644e581e3fb0_faked_filled.tsv.gz --ms-pol pos --names-precursor mz --names-rt rt

Usage: 
  fake_features_components.R [--fil-ann-pre=<fil-ann-pre>] [--fil-ann-fil=<fil-ann-fil>] [--fil-fea-raw=<fil-fea-raw>] [--ms-pol=<ms-pol>] [--names-precursor=<names-precursor>] [--names-rt=<names-rt>]

Arguments:
  fil-ann-pre                List of pretreated annotations. STRING
  fil-ann-fil                Filled annotations file. STRING
  fil-fea-raw                Raw features file. STRING

  ms-pol                     Polarity used. Must be "pos" or "neg". STRING

  names-precursor            Name of "precursor m/z" variable in the input. STRING
  names-rt                   Name of "retention time" variable in the input. STRING

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
