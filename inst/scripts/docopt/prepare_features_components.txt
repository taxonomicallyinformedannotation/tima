Usage: 
  prepare_features_components.R [--fil-ann-pre=<fil-ann-pre>] [--fil-ann-fil=<fil-ann-fil>] [--fil-net-spe-com-raw=<fil-net-spe-com-raw>] [--too-net-spe-com=<too-net-spe-com>]

Arguments:
  fil-ann-pre                List of pretreated annotations. STRING
  fil-ann-fil                Filled annotations file. STRING
  fil-net-spe-com-raw        Raw components file. STRING

  -h --help                  Shows this screen.
