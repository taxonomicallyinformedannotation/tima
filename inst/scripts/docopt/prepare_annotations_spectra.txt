You can use this script with the following example:
Rscript inst/scripts/prepare_annotations_spectra.R --fil-ann-raw-spe data/interim/annotations/example_isdb.tsv.gz --fil-ann-pre data/interim/annotations/example_isdb_prepared.tsv.gz

Usage:
  prepare_annotations_spectra.R [--fil-ann-raw-spe=<fil-ann-raw-spe>] [--fil-ann-pre=<fil-ann-pre>]

Arguments:
  fil-ann-raw-spe            List of spectral matching results. STRING
  fil-ann-pre                List of prepared annotations. STRING

Options:
  -h --help                  Shows this screen.
  -v --version               Shows version.
